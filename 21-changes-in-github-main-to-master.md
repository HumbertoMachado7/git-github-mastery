# Cambios en GIT HUB de Main a Master

Por cuestiones de inclusi√≥n, en **git hub** cambiaron el nombre de Master a Main.

Para corregirlo en **git bash**, hay que hacer los siguientes pasos.

Lo primero que tenemos que hacer para cambiar la rama principal de master a main, tenemos que posicionarnos en la carpeta del proyecto en nuestro equipo local y escribir lo siguiente.

git branch -m main

![alt text](<Images/Untitled 84.png>)

y despues para finalizar tenemos que hacer lo siguiente:

git push origin main
